# Mental Health Gita AI Companion

## Voice Cloning Feature
The Voice Cloning feature allows users to replicate any voice, enabling a more personalized interaction with the Mental Health Gita AI Companion. This feature utilizes state-of-the-art deep learning techniques to clone voices accurately. 

### How It Works:
1. **Data Collection**: Users provide audio samples of the voice that they wish to clone.
2. **Training**: The AI processes the audio samples to build a voice model.
3. **Integration**: The cloned voice can be integrated into the AI interactions, allowing for a human-like engagement.

## Language Composition Breakdown
Our AI Companion supports multiple languages, providing a comprehensive breakdown of sentence structure, grammar, and vocabulary usage in each language. The following languages are supported:
- English
- Spanish
- French
- Hindi

Each language has been rigorously tested to ensure accuracy and compatibility with our voice cloning capabilities.

## Updated Tech Stack
- **Frontend**: React.js

- **Voice Cloning**: TensorFlow and PyTorch

- **Hosting**: AWS

With the addition of the Voice Cloning feature, users can enjoy a more tailored interaction. The tech stack has been chosen to ensure scalability and efficiency, making the Mental Health Gita AI Companion more robust and user-friendly.
