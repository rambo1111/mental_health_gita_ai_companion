Installation Guide
1
Step — 01
Download
the App
Download and run the Windows installer. It installs the desktop interface on your machine.

Run the .exe and follow the on-screen prompts. No configuration needed.
Download Installer

2
Step — 02
Download
AI Files
Download the AI bundle. It contains the model weights, LoRA adapters, the AI backend, and the embeddings file — everything needed to run offline.
Download AI Bundle

3
Step — 03
Arrange
the Folder
Extract the zip. Then copy gita-companion.exe from the install location into the same folder. The final folder must look like this:
hf_cache\
lora_adapters_LLM1\
lora_adapters_LLM2\
lora_adapters_LLM3\
lora_adapters_LLM6\
gita_ai.exe
gita-companion.exe
router_embeddings.pkl
4
Step — 04
Launch
the App
Double-click gita-companion.exe from inside the folder you assembled. The AI models load in 30–60 seconds.

The app runs entirely offline. No internet required after setup.
